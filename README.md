# JogoDaVelhaPOO
Uma versão melhorada do meu primeiro projeto do jogo da velha. Apliquei conceitos de herança, polimorfismo e melhorias de código que fui aprendendo. Pretendo melhorar o projeto ao longo do tempo.

O código main está na classe "Aplicacao", responsável por controlar parte do fluxo do menu.
A classe "JogoDaVelha", funciona como a "mente" de todo o projeto, tem todos os métodos necessários para a validação e é responsável por rodar o jogo.
